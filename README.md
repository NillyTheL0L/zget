## Install

### 1. Download the installer

```http
  $ wget https://raw.githubusercontent.com/NillyTheL0L/zget/main/zgetBuild
```

| Name   | Type       | Description                           |
| :---------- | :--------- | :---------------------------------- |
| `zgetBuild` | `Script` | **Install zget** |

#### 2. Give permissions

```http
  $ chmod +x zgetBuild
```

#### 3. Install

```http
$ ./zgetBuild
```
#### 4. Execute Zget

```http
$ zget
```
| Name   | Type       | Description                           |
| :---------- | :--------- | :---------------------------------- |
| `zget` | `Script` | **Zget Script** |
